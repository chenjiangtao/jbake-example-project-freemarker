## jbake 常用命令
```sh
编译
jbake -b

运行，**一定要加上 -b ，不然上传文件时会导致崩溃！**

nohup jbake -b -s &

启动后不可再运行jbake -b !!! 

查找杀掉进程
ps aux | grep jbake | grep -v grep

kill -9 pid
```
