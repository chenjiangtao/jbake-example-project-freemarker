下面这种做法是错误的，不要试图让nginx直接指向

jbake -b -s /root/project /usr/local/nginx/html/jbake_output

nginx
  listen 80;

正确的办法是：


编译
jbake -b

运行
nohup jbake -b -s &

查找杀掉进程
ps aux | grep jbake | grep -v grep
